데이터 과학 – 시각화

AI is a Superpower

“AI is a superpower!!!”, 인공지능을 체득하면 슈퍼파워를 손에 쥘 것이다. Andrew Ng

금수저, 은수저 슈퍼파워를 받은 사람과 기계학습을 통달한 흑수저들간의 무한경쟁이 드뎌 시작되었다. 물론, 금수저를 입에 물고 기계학습을 통달한 사람이 가장 유리한 출발을 시작한 것도 사실이다.

학습목차

  • 시각화(Visualization)
    • 기념비적인 시각화 역작모음
    • 행복한 시각화 생활을 위한 비밀
    • R 그래프/코딩 글꼴(font)
    • R 시각화 색상(color)
    • 그래프 한장에 그래프 다수 찍기
    • 그림을 파일에 저장
    • 시각화 픽셀 정보량 극대화
  • 그래프 문법 실무
    • 그래프 문법 - ggplot
    • ggplot2 색상 제어
    • 데이터 시각화(ggplot2, ggvis)
    • lattice - 격자그림(Trellis Graphics)
    • 점그래프 - 전후변화
    • 단변량/다변량 시계열 데이터
    • 제19대 국회 본회의 출석부 - joyplot
    • 남북한 군사력 비교 - 와플(Waffle)
    • 2018년 달력
  • 범주형 데이터 시각화 기초
    • 범주형 자료 - 모자이크(mosaic): 통계청 출산(2016)
    • 발전 에너지(원자력 포함) - 대응분석(Correspondance Analysis)
  • 애니메이션(Animation)
    • 동영상 그래프(애니메이션) - 정당지지율
    • 왜 아프리카인가?
    • 새해 복 많이 받으세요: 2020년 쥐띠
    • 빅4 회계법인 분석
    • gif 이미지 편집
  • 인터랙티브(Interactive) 시각화
    • 지진해일 대피소 - crosstalk
    • ggvis = 웹브라우져 기능(Vega/D3) + ggplot
    • 북한 미사일 발사 연대기
    • 북한 미사일 발사 분석
    • 무역보복 - UN Comtrade API
    • rbokeh vs plotly
      • rbokeh + trelliscope
      • plotly + ggplot2
      • 국회의원 사진 - trelliscopeJS
        • 빅데이터 시각화 - trelliscopeJS
        • 빅데이터 시각화 주식 - trelliscopeJS
  • 시각화 실무
    • 세계보건기구(WHO) - 전문시각화
    • 가구당 가계수지 - 10 분위별
    • 러시아 월드컵(2018) - 한국 vs 독일
    • treemap
      • 대한민국 인구변화(2000 vs 2015) - treemap
      • 지역내총생산 (GRDP) - treemap
    • 유통소매점 - 인터랙티브 heatmap
    • 저출산과 소요예산 - 상관관계
    • 이모지(emoji): 코로나19
